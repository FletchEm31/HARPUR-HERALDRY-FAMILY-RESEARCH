================================================================
PURPOSE OF THIS FILE
================================================================
This file defines how theory is handled in the Harper System.
Theory is not the enemy of good genealogy. Unflagged theory is.

================================================================
WHAT BELONGS IN ANNEX B
================================================================
  - Hypotheses about chain gaps not yet closed by primary source
  - Logical inferences from circumstantial evidence
  - Working leads not yet confirmed
  - Aggregator source data flagged as unverified
  - Any statement beginning with "likely", "probably", "possibly"

================================================================
HARD RULES FOR THEORY LAYER
================================================================
1. Theory never enters MAIN DOC under any circumstances
2. Theory must be labeled with evidence level (LOW/MEDIUM/HIGH)
3. Theory must state explicitly what document would confirm it
4. Confirmed theory moves to ANNEX A via PROMOTE_TO_MAIN_DOC
5. Disproven theory is retained in ANNEX B with DISPROVEN flag
   (Do not delete — disproven leads prevent future re-investigation)

================================================================
THE ANNEX B DISCIPLINE
================================================================
A well-maintained ANNEX B is a sign of rigorous research.
It shows the College of Arms that you separated fact from theory
and did not contaminate your primary chain with assumption.
Present it openly. It is a strength, not a weakness.
