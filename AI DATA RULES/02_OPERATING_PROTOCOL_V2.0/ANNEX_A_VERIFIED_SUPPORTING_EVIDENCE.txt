================================================================
PURPOSE
================================================================
Evidence that directly supports MAIN DOCUMENT claims.
This is the evidence room. Everything here backs something
already in the confirmed chain or directly adjacent to it.

If it does not support a MAIN DOC entry — it does not belong here.
Move it to ANNEX C (collateral) or ANNEX B (theory).

================================================================
WHAT BELONGS HERE
================================================================
  - Parish register extracts confirming MAIN DOC individuals
  - Will and probate summaries for direct chain ancestors
  - Hearth Tax entries confirming MAIN DOC household
  - Visitation pedigree entries naming MAIN DOC individuals
  - Chancery records naming MAIN DOC individuals
  - IPM records for MAIN DOC ancestors
  - Secondary source confirmations (Tier 2) of MAIN DOC entries
    [flagged as SECONDARY — must be verified against primary]

================================================================
WHAT DOES NOT BELONG HERE
================================================================
  - Collateral family members not in the direct line
  - Surname clusters not linked to a MAIN DOC individual
  - Theory or hypothesis about unconfirmed connections
  - Aggregator source data (Ancestry trees, unsourced FamilySearch)
  - Any document not explicitly referencing a MAIN DOC individual

  Certainty:       [PRIMARY CONFIRMED / CONFIRMED / LIKELY / POSSIBLE]

  Notes:           [Any relevant detail from the document]

================================================================
CERTAINTY LABELS (mandatory — no unlabeled entries)
================================================================
  PRIMARY CONFIRMED     Original document image confirmed. States
                        the fact directly. Can enter MAIN DOC.

  CONFIRMED             Primary source exists. Fact inferred from
                        clear context. Strong but not direct.

  LIKELY                Tier 2 source. Primary not yet confirmed.
                        Cannot enter MAIN DOC without upgrade.

  POSSIBLE              Tier 3 index only. Document not yet seen.
                        Flag for follow-up research.

  UNCONFIRMED           Lead only. No source located yet.

  DISPROVEN             Investigated and eliminated. Retained to
                        prevent future re-investigation of dead end.
